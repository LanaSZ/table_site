Підключення проекту до репо

  • git init - ініціалізація проекту
  • git add . - додати усі файли
  • git commit -m “first commit” - додаємо коментраій до наших змін
  • **git remote add origin https://github.com/konstantine811/Table_Site.git** - підключаємо проект до віддаланого репозиторію
  • git push -u origin master - заливаємо проект на репозіторій
